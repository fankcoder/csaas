# cs_web 提示词

## 模块定位

`cs_web/` 是项目的前端工作台，负责品牌呈现、SaaS 转化、套利列表展示、个人中心、内容页面和与后端 API 的交互。

技术栈：

- `Next.js 14`
- `React 18`
- `TailwindCSS`

## 前端目标

- 用 SaaS 风格把复杂套利数据包装成用户易理解的产品
- 首页需要兼顾 CS 游戏氛围与国际化 SaaS 质感
- 每个主要能力独立成页，避免单页堆砌
- 兼顾 SEO、转化、权限引导和数据消费

## 目录映射

```text
cs_web/src/
  app/         页面与路由
  components/  通用布局与展示组件
  lib/         API、认证、本地辅助工具、内容数据
```

## 页面分层

- 营销与 SEO 页面：首页、FAQ、Blog、Guide、法律页
- 转化页面：登录、定价、样例榜单
- 功能页面：套利列表、单饰品详情、个人中心、收藏、收益报表、开通清单

## 联调规则

- 默认 API 基地址从 `NEXT_PUBLIC_API_BASE_URL` 读取
- 需要登录的页面通过本地 token 驱动
- 后端错误信息要直接反馈给用户，而不是吞掉

## UI/UX 原则

- 不做普通模板站，要有明确品牌感
- 首页视觉要更“CS2 数据战情室”，而不是泛企业后台
- 内页则回到更克制的 2026 SaaS 风格，保证信息密度与可读性
- 所有数据列表要考虑分页、空状态、未授权状态、加载状态
